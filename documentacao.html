Passando o cursor do mouse por cima de algum texto você pode ver uma breve descrição do termo. Você pode encontrar em:
  * Nos parâmetros opcionais ou parciais (em itálico)
  * Na versão de publicação
  * Nas tags de um parâmetro (na lista)
  * Na função, nos exemplos
  * Nos parâmeros da função, nos exemplos
  * Em palavras sublinhadas de descrições

COMO USAR CADA FUNÇÃO

randomize(Max, Min, Dec) - Desde v1.0.0

Peça um número aleatório entre dois extremos. Escolha se quer casas decimais ou não. Você também pode sortear elementos de um vetor
  • [Int] Max: Valor máximo a ser sorteado OU vetor com os elementos a serem sorteados
  • [Int] (Opc) Min: Valor mínimo a ser sorteado
  • [Int] (Opc) Dec: Quantidade de casas decimais no resultado
     

 

chance(Valor, Chance, Acompanhar) - Desde v1.1.0

Sorteie um elemento de um vetor trabalhando em cima de chance
  • [Array] Valor: Vetor onde estão os elementos a ser sorteado
  • [Array] Chance: Vetor onde estão as chances de sorteio de cada elemento
  • [Bool] (Opc) Acompanhar: Se desejas receber mensagens do progresso do sorteio no terminal
     

Exemplo
chance(['A', 'B', 'C', 'D', 'E'], [1, 3, 2, 5, 3], false)
O sistema irá sortear um valor aleatório, irá ver qual posição caiu nesse peso e retornar o elemento dessa posição
 

avg(Vetor, Dec) - Desde v1.0.0

Tire uma média dos valores que estão dentro do vetor. Valores não numéricos serão ignorados
  • [Array] Vetor: Vetor com os valores dentro
  • [Int] (Opc) Dec: Quantidade de casas decimais no resultado
     

Exemplo
avg([1, 4, 3, 6, 'C', 5, 4, 1], 1)
O sistema irá remover os elementos que não for número e tirar uma média de todos os valores restantes
 

sequencia(Max, Min, Espaco) - Desde v1.1.0

Cria um vetor com números do Min ao Max pulando uma quantidade dada em Espaco
  • [Int] Max: Valor máximo que deve ter na sequência
  • [Int] (Opc) Min: Valor mínimo que deve ter na sequência
  • [Dec] (Opc) Dec: Qual o espaço deve ter entre cada número da sequência
     

Exemplo
sequencia(15, 0, 2)
O sistema irá ver a distância escolhida, criar a sequencia respeitando a distancia e parar quando chegar no limite
 

repete(Valor, Local) - Desde v1.0.0

Verifique quantas vezes tal valor apareceu em um conjunto
  • [Any] Valor: Qual o valor (ou texto) deve ser verificado
  • [Array] Local: Local onde deve ser verificado
     

Exemplo
repete(3, [1, 5, 3, 4, 3, 'Zienaps', 5, 3, 3, 2, 1])
O sistema verifica quantos elementos tem o mesmo valor do valor procurado e trás quantos encontrou
 

sortear(Tamanho, Quantidade, Repete) - Desde v1.0.0

Crie um vetor com números sorteados, repetidos ou não
  • [Int] Tamando: Qual o valor máximo a ser sorteado
  • [Int] (Opc) Quantidade: Quantos números devem ser sorteados
  • [Bool] (Opc) Repete: Se os números sorteados podem repetir
     

Exemplo
sortear(30, 3, false)
O sistema sorteia um número do vetor inicial e, se false, remove-o da lista. Repete o processo várias vezes e depois traz em vetor o resultado
 

unificar(Vetor, Acompanhar) - Desde v1.0.0

Remova valores duplicados de seu vetor permanecendo somente o último elemento repetido
  • [Array] Vetor: Vetor com os elementos a serem verificados
  • [Bool] (Opc) Acompanhar: Se deve trazer em log o processo de verificação (Isso pode atrasar)
     

Exemplo
unificar([1, 6, 5, 2, 3, 4, 3, 2, 5, 1, 3], false)
O sistema remove os termos que ele verificar se já existe após ele
 

filtragem(Onde, oQue, Manter, Acompanhar) - Desde v1.0.0

Veja se os valores de um vetor estão no segundo vetor e mantenha os que não tiver, ou vice-versa
  • [Array] Onde: Vetor a verificar os valores. Retornará esse vetor filtrado pelo oQue
  • [Array] oQue: Vetor onde o sistema usará de base para filtrar o primeiro vetor
  • [Bool] (Opc) Manter: true caso queira manter somente os valores encontrados
  • [Bool or 'Impo'] (Opc) Acompanhar: Se deve trazer em log o processo de verificação (Isso pode atrasar a resposta). 'Impo' mostra somente logs importantes
     

Exemplo 1 (remover)
filtragem([1, 3, 5, 7, 4, 8, 4, 3, 2, 10, 2, 5], [3, 5, 8])
O sistema verifica se o valor de certo elemento está contido no segundo vetor. Se sim, ele é removido do vetor principal
 
Exemplo 2 (manter)
filtragem([1, 3, 5, 7, 4, 8, 4, 3, 2, 10, 2, 5], [3, 5, 8], true)
O sistema verifica se o valor de certo elemento não está contido no segundo vetor. Se não estiver, ele é removido do vetor principal
 

mesTamanho(Mes, Ano) - Desde v1.2.0

Traga a quantidade de dias que tem em tal mês
  • [Int] Mes: Qual mês quer buscar o tamanho
  • [Int] (Par) Ano: De qual ano quer buscar o dia de tal mês
     

Exemplo
mesTamanho(5)
O sistema irá procurar em uma lista o número do mês buscado. Ele irá retornar o valor correspondente
 

mesNome(Mes, Tipo, Língua) - Desde v1.2.0

Traga o nome de um mês em um formato e língua específica
  • [Int] Mes: Qual mês quer bucar o nome
  • [Str] (Opc) (Res) Tipo: Como retornar o mês
  • [Str] (Opc) (Res) Lingua: Em qual idioma deve trazer o resultado
     

Exemplo
mesNome(5, 'Tres')
O sistema irá buscar o mes em uma lista, separar a língua pedida e depois o tipo pedido
 

dataBuscar(Data, Busca) - Desde v1.2.0

Traga um valor específico de uma data
  • [Date] Data: Data que queira verificar
  • [Str] (Res) Busca: Valor que queira buscar
    • data: Retorna a data completa
    • dia: Retorna o dia em número
    • semana: Retorna o dia da semana em número
    • mes: Retorna o mês em número
    • mess: Retorna o mês em texto
    • ano: Retorna o ano em número
    • hoje: Retorna a data sem horário
    • hora: Retorna a hora em número
    • minuto: Retorna o minuto em número
    • segundo: Retorna o segundo em número
    • milissegundo: Retorna o milissegundo em número
    • atualMilissegundo: Retorna a quantidade de milissegundos desde 01/01/1970 00:00
    • atualSegundo: Retorna a quantidade de segundos desde 01/01/1970 00:00
    • atualMinuto: Retorna a quantidade de minutos desde 01/01/1970 00:00
    • atualHora: Retorna a quantidade de horas desde 01/01/1970 00:00
    • atualDia: Retorna a quantidade de dias desde 01/01/1970 00:00

     

Exemplo
dataBuscar([2020, 5, 9], 'dia')
O sistema irá ler a data (ou transformar o vetor [yyyy, mm, dd] em data, depois irá buscar o valor que procura
 

[EXCLUSO] LimparLog(Aviso, Tamanho) - Desde v1.0.0

Limpe o terminal
  • [Bool] (Opc) Aviso: Se deve ser avisado quando a limpeza for concluída
  • [Int] (Opc) Tamanho: Quantas linhas deve ser adicionadas
     

Exemplo
LimparLog(true, 20)
O sistema envia uma quantidade de linhas vazias para esconder as mensagens
 

[EXCLUSO] HowToUse(Descrição, Significado, Exemplo) - Desde v1.0.0

Veja como usa cada function
  • [Bool] (Opc) Descrição: Se quer ver a decrição das funções
  • [Bool] (Opc) Significado: Se quer ver o significado de cada parâmetro
  • [Bool] (Opc) Exemplo: Se quer ver exemplos de cada função
     

Exemplo
HowToUse(true, true, false)
O sistema irá enviar mensagens no terminal explicando o uso de cada função. Será mostrado somente os parâmetros marcados como true
 

[EXCLUSO] Checar(Tipo) - Desde v1.2.0

Receba uma notificação no terminal sempre quando o código alcançar tal ponto
  • [Str] (Opc) Tipo: Qual mensagem enviar
    • 'inicio': Mensagem pré-definida de início de código
    • 'fim': Mensagem pré-definida de conclusão de código
    • Outro texto: Mensagem pré-definida com um Checkpoint de identificação própria

     

Exemplo
Checar('inicio')
O sistema irá enviar uma mensagem pré-definida no terminal de acordo com o tipo de notificação
 

[EXCLUSO] Testar(Exclusivos) - Desde v1.2.0

Teste todas as funções ou algumas específicas. Adicione-os em um vetor e mande para a função
  • [Array] (Opc) (Res) Exclusivos: Marque, como String, quais funções quer testar
     

Exemplo
Testar(['randomize', 'sortear'])
O sistema irá testar funções citadas (ou todas, se não citado), utilizando alguns valores já definidos
